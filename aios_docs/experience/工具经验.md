# 工具经验

## 经验 1：LLM API Key 不写入配置文件

经验等级：A

适用场景：所有需要调用外部 LLM 的项目。

不适用场景：无外部 API 调用的离线项目。

问题表现：API Key 写入配置文件，存在泄露风险。

原因分析：配置文件可能被复制、上传、提交到仓库。

解决方案：API Key 从环境变量读取。

以后规则：`aios_config.yaml` 只写 `api_key_env`，不写真实 key。

置信度：高

来源项目：AIOS 工具设计。

日期：2026-06-05

是否可能过期：否

过期检查方式：长期有效。

## 经验 2：Codex 非交互调用使用 `codex exec`

经验等级：A

适用场景：Python 调用 Codex CLI 执行任务。

不适用场景：用户手动 TUI 对话。

问题表现：Python 调用交互式 Codex 难以控制输出和状态。

原因分析：交互式会话不适合自动化调度。

解决方案：使用 `codex exec --json --cd <源码目录> "任务"`。

以后规则：AIOS 自动调用 Codex 默认走 `codex exec`。

置信度：高

来源项目：AIOS 工具设计。

日期：2026-06-05

是否可能过期：是

过期检查方式：如果 Codex CLI 参数变化，需要重新验证。
